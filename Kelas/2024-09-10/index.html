Belajar HTML Dan CSS

HTML digunakan untuk membuat tampilan website. CSS digunakan untuk mengetur tampilan website

Materi HTML

  1. h1 untuk judol
  2. p untuk paragraf
  3. img untuk gambar
  4. table untuk tabel
  5. input untuk form
  6. li untuk list/daftar

Materi CSS

  • margin, padding, position untuk mengatur jarak antara elemen HTML
  • display untuk mengatur elemen html agar tampil atau tidak tampil, mendatar, atau menurun
  • style untuk mengatur font
